# Hangman Game (Python)

A simple text-based Hangman game developed using Python as part of the CodeAlpha Python Programming Internship (Task 1).

## Features
- Console-based gameplay
- 3 difficulty levels (Easy, Medium, Hard)
- Maximum 6 incorrect guesses
- Hint system
- Score tracking

## Concepts Used
- Random module
- While loops
- If-else conditions
- Lists and strings
- Functions

## How to Run
1. Make sure Python is installed
2. Run the following command:
   python hangman.py
